# Go Load Balancer

![Go Version](https://img.shields.io/badge/Go-1.25-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
[![CI](https://github.com/yourusername/go-load-balancer/workflows/CI/badge.svg)](https://github.com/yourusername/go-load-balancer/actions)

A high-performance HTTP/HTTPS load balancer written in Go with multiple load balancing algorithms, health checks, and Docker support.

## Features

- ✅ **Multiple Load Balancing Algorithms**
  - Round Robin
  - Least Connections
- ✅ **Automatic Health Checks**
- ✅ **Concurrent Request Handling**
- ✅ **Docker & Docker Compose Support**
- ✅ **Zero Dependencies** (uses only Go standard library)
- ✅ **Production-Ready**
